requirements to run this project 

NodeJS, Python3

install python/django dependencies: 
cd into r8scholar/ 
run the following line 
pip install Django==3.1.5 djangorestframework==3.12.2 django-cors-headers==3.7.0

install NodeJS if you dont have it 
https://nodejs.org/en/
then cd into r8scholar/frontend and run 
npm init -y this will install the node_modules folder and initialize a node project 
then run 

npm install --save-dev babel-core babel-loader babel-plugin-transform-class-properties babel-preset-es2015 babel-preset-react css-loader style-loader url-loader file-loader webpacknpm install --save-dev @babel/core
npm install --save-dev @babel/core @babel/env @babel/preset-env @babel/preset-react @babel/plugin-transform-runtime

then 

npm install --save react react-dom bootstrap react-bootstrap

then 

npm i react-icons react-icons-kit react-star-rating react-cookies


to run the program cd into /r8scholar (the directory with manage.py) 

!!!!!FIRST TIME RUNNING THE SERVER!!!!!!
-Migrate the database 
python3 manage.py makemigrations 
python3 manage.py migrate 

and type: python3 manage.py runserver 

to bundle the javascript open a new terminal window cd into /r8scholar/frontend and type:
npm run build

this puts webpack into watch mode, it will bundle the javascript then continue to 'watch' for updated files to give a live update
